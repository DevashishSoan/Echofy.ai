# Echofy.ai

Echofy.ai is a real-time voice-to-text web app designed for creators, professionals, students, and accessibility users. It offers fast (<100ms latency), highly accurate, multilingual transcription with tools for editing, exporting, and collaboration. Built using React for the frontend, Supabase for backend/auth, and a custom speech engine. Echofy.ai aims to make seamless, real-time transcription accessible to everyone.

## Features
- Instant transcription with <100ms latency
- High accuracy, multilingual support
- Transcript editing & export tools
- Collaboration-ready interface

## Tech Stack
- Frontend: React
- Backend: Supabase
- Speech Engine: Custom-built

## Getting Started
Clone the repo, install dependencies, and run:
```bash
yarn install
yarn dev
```
